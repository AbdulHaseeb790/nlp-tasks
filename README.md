# NLP Tasks 🤖

A collection of NLP tasks built using HuggingFace Transformers and Python.

## Projects

### 1. Text Summarization
Summarizes long text into short, meaningful summaries using HuggingFace pipeline.

### 2. Text Translation
Translates text between languages using HuggingFace pipeline.

### 3. Named Entity Recognition (NER)
Detects and classifies named entities (people, places, organizations) in text.

## Tech Stack
- Python
- HuggingFace Transformers
- Google Colab
